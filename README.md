# rabbitmq-http-api-client-zig

A Zig client for the [RabbitMQ management HTTP API](https://www.rabbitmq.com/docs/management).

Modeled after [rabbitmq-http-api-client-rs](https://github.com/michaelklishin/rabbitmq-http-api-client-rs) (Rust).


## Quick Start

```zig
const std = @import("std");
const api = @import("rabbitmq_http_api_client");

pub fn main() !void {
    const allocator = std.heap.page_allocator;
    var threaded_io: std.Io.Threaded = .init(allocator, .{});
    defer threaded_io.deinit();

    var client = try api.Client.init(allocator, threaded_io.io(), .{});
    defer client.deinit();

    const overview = try client.getOverview();
    defer overview.deinit();
    std.debug.print("RabbitMQ {s}\n", .{overview.value.rabbitmq_version.?});

    const reachable = client.probeReachability();
    std.debug.print("Reachable: {}\n", .{reachable.successful});
}
```


## TLS

Pass an absolute path to a [CA certificate bundle](https://www.rabbitmq.com/docs/ssl#peer-verification)
in the PEM format when constructing the client.
This is required for self-signed certificates such as those generated by `tls-gen`.

```zig
var client = try api.Client.init(allocator, io, .{
    .endpoint = "https://localhost:15671/api",
    .ca_cert_file = "/path/to/ca_certificate.pem",
});
defer client.deinit();
```


## Listing Connections

```zig
const conns = try client.listConnections();
defer conns.deinit();
for (conns.value) |c| {
    std.debug.print("{s}\n", .{c.name});
}
```


## Closing Connections

```zig
const conns = try client.listConnections();
defer conns.deinit();
for (conns.value) |c| {
    try client.closeConnection(c.name, "test cleanup", false);
}

try client.closeUserConnections("guest", "test cleanup", true);
```


## Declaring Queues, Exchanges, and Bindings

```zig
try client.declareClassicQueue("/", "my.queue");
try client.declareQuorumQueue("/", "my.qq");
try client.declareStream("/", "my.stream");

try client.declareFanoutExchange("/", "my.fanout");
try client.bindQueue("/", "my.fanout", "my.queue", .{ .routing_key = "" });
```


## Policies

```zig
var def = api.builders.PolicyDefinitionBuilder.init(allocator);
defer def.deinit();
const definition = try def.maxLength(1000).deadLetterExchange("dlx").build();

try client.declarePolicy("/", "size-limit", .{
    .pattern = "^logs\\.",
    .definition = definition,
    .@"apply-to" = "queues",
});
```
